# Emonet Lab

Research in collective behavior, sensory coding, and navigation.

Website: [emonet.biology.yale.edu](http://emonet.biology.yale.edu/)

## Selected public repositories

- [emonetlab/opto-track](https://github.com/emonetlab/opto-track): closed-loop fly tracking and fictive odor stimulation experiments.
- [emonetlab/kontroller](https://github.com/emonetlab/kontroller): MATLAB tools for NI-DAQ-driven experiments and automation.
- [emonetlab/B-FRET](https://github.com/emonetlab/B-FRET): Bayesian inference for extracting FRET signals from noisy measurements.
- [emonetlab/nfsim](https://github.com/emonetlab/nfsim): stochastic biochemical reaction simulation for large rule-based networks.
- [emonetlab/microbetracker](https://github.com/emonetlab/microbetracker): bacterial cell segmentation and fluorescence analysis in microscopy images.
- [emonetlab/controlling-dynamic-stimuli](https://github.com/emonetlab/controlling-dynamic-stimuli): code accompanying [*Controlling and measuring dynamic odorant stimuli in the laboratory*](https://jeb.biologists.org/content/early/2019/11/06/jeb.207787.abstract).
- [emonetlab/bifurcation-temporal-information](https://github.com/emonetlab/bifurcation-temporal-information): code accompanying [*Bifurcation Enhances Temporal Information Encoding in the Olfactory Periphery*](https://journals.aps.org/prxlife/abstract/10.1103/PRXLife.2.043011).
- [emonetlab/nongenetic-adaptation-wave](https://github.com/emonetlab/nongenetic-adaptation-wave): code accompanying [*Nongenetic adaptation by collective migration*](https://doi.org/10.1073/pnas.2423774122).
