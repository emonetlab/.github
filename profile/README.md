# Emonet Lab

Research in collective behavior, sensory coding, and navigation.

Website: [emonet.biology.yale.edu](http://emonet.biology.yale.edu/)

## Selected public repositories

- [emonetlab/emonetlab](https://github.com/emonetlab/emonetlab): lab website source.
- [emonetlab/opto-track](https://github.com/emonetlab/opto-track): closed-loop fly tracking and fictive odor stimulation experiments.
- [emonetlab/kontroller](https://github.com/emonetlab/kontroller): MATLAB tools for NI-DAQ-driven experiments and automation.
- [emonetlab/B-FRET](https://github.com/emonetlab/B-FRET): Bayesian inference for extracting FRET signals from noisy measurements.
- [emonetlab/nfsim](https://github.com/emonetlab/nfsim): stochastic biochemical reaction simulation for large rule-based networks.
- [emonetlab/microbetracker](https://github.com/emonetlab/microbetracker): bacterial cell segmentation and fluorescence analysis in microscopy images.
- [emonetlab/controlling-dynamic-stimuli](https://github.com/emonetlab/controlling-dynamic-stimuli): code accompanying the dynamic odorant stimuli work in the *Journal of Experimental Biology*.
- [emonetlab/bifurcation-temporal-information](https://github.com/emonetlab/bifurcation-temporal-information): code for the manuscript on bifurcation and temporal information encoding in the olfactory periphery.
- [emonetlab/nongenetic-adaptation-wave](https://github.com/emonetlab/nongenetic-adaptation-wave): reproduction scripts for the manuscript *Nongenetic adaptation by collective migration*.
- [emonetlab/SensoryDiversityTuningAnalysis](https://github.com/emonetlab/SensoryDiversityTuningAnalysis): analysis code accompanying *Orthogonal tuning of sensory diversity across multiple stimuli*.

Repository index: [github.com/orgs/emonetlab/repositories](https://github.com/orgs/emonetlab/repositories)
